webApp que é um Diário onde a pessoa pode escrever seus sentimentos ,registrando um histórico de entradas e permitindo acessar uma frase motivacional.

---
## 📚 Aprendizados
- Processamento de linguagem natural
- Uso de API
- DOM 
- JSON

## ⚙️ Funcionalidades
- Digite seu nome, data/hora, o titulo da entrada , e como você está se sentindo hoje.
- Clique no botão "salvar no Diário".
- É possível curtir e comentar nas postagens.
- Caso queira uma frase motivacional clique no botão abaixo.

## 🛠️ Stack

- JavaScript
- HTML5
- CSS3
- Web Speech API

## 🚀 Deploy
1. Clone ou baixe este repositório.

```bash
git clone https://github.com/LauraCirino/diario-de-emocoes
```

2. Abra o arquivo `index.html` em seu navegador.
